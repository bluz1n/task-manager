## Description

Task manager project using [Nest](https://github.com/nestjs/nest) The focus on this project is to understand and apply Nest core concepts, and add useful features to enhance it.

Using Repository pattern, ORM, Docker and authorization using JWT. 
Planning on using GraphQL for future stuff

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Database

This project uses [MikroORM](https://mikro-orm.io/) with a Postgres database running on a Docker container. 


## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
